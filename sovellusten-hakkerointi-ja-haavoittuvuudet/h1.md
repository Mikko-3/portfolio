# h1 Scope First, Controls Later

## a) Perustaso

Tehtävä: Laadi ISMS-soveltamisala (scope) omalle kotiverkollesi ja opiskelulabillesi (Iso-Anttila & Karvinen s.a.).

### Soveltamisala

**Perus infra:**

TP-Link reititin.

**Kurssiharjoituksiin käytettävät laitteet:**

Kannettava tietokoneeni ja siihen asennettu Linux virtuaalikone. Puhelin tarvittaessa MFA kirjautumisissa esim. Moodleen.

**Tiedot:**

Kurssimateriaalit Moodlessa, Teron verkkosivut, Github repositorio tehtäviä varten. Kirjautumistiedot Laksuun, Moodleen ja Githubiin.

### Rajaukset

Pöytäkoneeni ja operaattorin verkko. Operaattorin verkkoa ja internettiä en omista, enkä voi itse hallita.
En käytä pöytäkonettani koulutehtävien tekoon, joten se on merkityksetön kurssin näkökulmasta.

### Keskeiset rajapinnat

Pilvipalvelut GitHub ja Moodle.
Reititin kotiverkon ja internetin välisenä rajana.
Elisa Oyj internet-palveluntarjoajana.
Oracle Virtualbox virtualisointialustana.

### Scope-teksti

Tämä scope-teksti sisältää "Sovellusten hakkerointi ja haavoittuvuudet" -kurssin harjoitusympäristön soveltamisalan.
Soveltamisala sisältää kotonani tarvittavan infrastruktuurin, laitteet, rajapinnat ja käsiteltävät tiedot.
Teksti sisältää myös rajaukset harjoitusympäristön ulkopuolelle (out-of-scope).
Kurssilla käsitellään sovellusten hakkerointia ja haavoittuvuuksia, joten on mahdollista, että kurssin aikana kohtaa haitallista koodia tai arkaluontoista tietoa.
Määrittelemällä ympäristön, voidaan välttää haitallisen koodin tai arkaluontoisen tiedon leviäminen internettiin tai muihin lähiverkossa oleviin laitteisiin.

Ympäristöni infrastruktuuri koostuu TP-Link reitittimestä, joka toimii rajapintana internetin ja lähiverkkoni välillä.
Reitittimen asetuksiin pääsy on suojattu vahvalla salasanalla, sekä asetukset port forwarding ja UPnP on otettu pois päältä.
Reitittimeen yhdistyy kannettava tietokoneeni ja puhelimeni WLAN yhteydellä, joka käyttää WPA3-Personal suojausta.
Kannettavani on Lenovo Thinkpad L14 (1. gen.), johon on asennettu Windows 11 käyttöjärjestelmä.
Käytössä on Windowsin oma palomuuri ja virustorjuntaohjelmisto.
Kannettavalle on asennettuna virtuaalikone Linux Debian 13 Trixie -käyttöjärjestelmällä, jonka virtualisointialustana toimii Oracle Virtualbox.
Virtuaalikoneen verkkoyhteys on otettu pois käytöstä.
Ympäristössä käytetään GitHub ja Moodle pilvipalveluita, sekä Laksu sivustoa.



## b) Sidotaan standardiin

Tehtävä: Tunnista vähintään 2 interested party -tahoa kotiverkkokontekstissa ja kirjoita kullekin:

- tarve/odotus/vaatimus (security/privacy/availability tms.)
- mihin ISO 27001 -vaatimusalueeseen se kytkeytyy (konteksti / johtajuus / suunnittelu / toiminta / arviointi / parantaminen)
- miten osoittaisit sen täyttymisen (evidence)
